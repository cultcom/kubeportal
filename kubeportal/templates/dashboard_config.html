kubectl

The kubectl command-line tool uses this config file to find the information it needs to connect to the cluster.

Windows

  • Install kubectl for Windows
  • Navigate to your home directory: cd %USERPROFILE%
  • Create the .kube directory: mkdir .kube
  • Save the config file in this directory

MacOS X / Linux / Unix

  • Install kubectl with your favourite package manager
  • Navigate to your home directory: cd ~
  • Create the .kube directory: mkdir .kube
  • Save the config file in this directory

You can test your installation by calling kubectl cluster-info.
